Data Mining Project Shubham, a backend web-dev has successfully done this project.

This project is based on data mining and visualization.I took a sample data set consisting of email messages recorded over 6 yrs data of people communicating over a open source LMS called SAKAI. Then I refined it and stored it in a database and used d3.js for visualization.Here I used the word cloud.
